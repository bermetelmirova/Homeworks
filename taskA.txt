Класс String имеет модификатор final
1. Можно передавать строку между потоками и не беспокоиться что она будет изменена
2. Отсутствие утечек памяти
3. В Java строки используются для передачи параметров для авторизации, открытия файлов и т.д. - неизменяемость 
позволяет избежать проблем с доступом

Это  позволяет экономить память и не создавать новые объекты для каждой повторяющийся строки. В случае с
изменяемыми строками - изменение одной приводило бы к изменению всех строк одинакового содержания.
